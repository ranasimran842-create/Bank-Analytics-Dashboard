# Bank Analytics Dashboard

## 📌 Project Overview
Bank Analytics Dashboard is a data analysis project built using SQL, MySQL, and Power BI. 
The project focuses on analyzing customer data, loan details, and transaction records to generate meaningful business insights.

## 🛠️ Tools & Technologies
- MySQL
- SQL
- Power BI
- CSV Data

## 📊 Dashboard Features
- Customer analysis
- Loan amount analysis
- Transaction insights
- Data visualization using Power BI dashboards
- Business insights from banking data

## 📂 Dataset
The project contains:
- customers.csv
- loans.csv
- transactions.csv

## 🎯 Objective
To analyze banking data and create an interactive dashboard that helps understand customer behavior, loan trends, and transaction patterns.
